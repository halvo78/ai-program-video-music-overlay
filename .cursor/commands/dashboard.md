# /dashboard Command

Manage Next.js dashboard.

## Usage

```
/dashboard [action]
```

## Actions

| Action | Description |
|--------|-------------|
| `start` | Start Next.js dev server |
| `stop` | Stop Next.js server |
| `restart` | Restart Next.js server |
| `status` | Show server status |
| `build` | Build for production |

## Examples

```bash
# Start dashboard
/dashboard start

# Stop dashboard
/dashboard stop

# Restart dashboard
/dashboard restart

# Check status
/dashboard status

# Build for production
/dashboard build
```

## Server Information

- **Port**: 3000 (default)
- **URL**: http://localhost:3000
- **Dev Mode**: Hot reload enabled
